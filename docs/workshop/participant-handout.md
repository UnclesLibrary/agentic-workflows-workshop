## 🎯 Learning Progression

```
Setup          → Install tools, create repository, configure secrets
Exercise 1     → Simple workflow (daily status) via CLI + Cloud Agent
Exercise 2     → Manual workflow creation (CI Coach)
Exercise 3     → Manual workflow creation (CI Doctor)
Exercise 4     → Cloud Agent workflow (test updates)
Exercise 5     → VS Code Repository Agent
Exercises 6-7  → Community workflow reuse (triage + planning)
Exercise 8     → Open exploration
```

You'll experience three different approaches to creating workflows, each with increasing quality and sophistication:

1. **Interactive CLI** (`gh aw new`) - Quick and interactive, good for learning
2. **Cloud Agent** (web browser agent tab) - Better quality by referencing documentation
3. **Repository Agent** (VS Code Copilot Chat after `gh aw init`) - Best quality with repository context
